# prompt-improver

## Overview

This skill improves user prompts so they produce clearer, more reliable, and more structured outputs from AI systems. It rewrites prompts using best practices such as adding context, specifying goals, defining output format, and reducing ambiguity.

## Instructions

### 1. Understand the Original Prompt

Read the user's prompt and determine:
- The user's main objective
- The task type (generation, coding, research, summarization, etc.)
- Any missing context or unclear instructions

### 2. Identify Issues

Check the prompt for common problems such as:
- Lack of context
- Vague instructions
- Missing output format
- Multiple tasks in one prompt
- Ambiguous wording

### 3. Improve the Prompt

Rewrite the prompt to make it more effective by:
- Adding clear instructions
- Including necessary context
- Defining the expected output format
- Breaking complex tasks into steps
- Making the task explicit

### 4. Return the Improved Prompt

Provide:
1. The improved prompt
2. A short explanation of what was improved

## Output Format

Return the response in the following structure:

Improved Prompt:
<rewritten prompt>

What Was Improved:
- improvement 1
- improvement 2
- improvement 3
